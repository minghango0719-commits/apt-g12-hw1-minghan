### 題目說明

請使用 Python 撰寫一個經典的「猜數字遊戲」。程式的邏輯如下：

1.  程式會在 1 到 100 之間，隨機產生一個秘密數字。
2.  玩家可以重複輸入猜測的數字。
3.  每一次猜測後，程式會根據結果給予提示：
    *   如果猜測的數字比秘密數字大，則印出「太大了！」。
    *   如果猜測的數字比秘密數字小，則印出「太小了！」。
    *   如果猜中了，則印出「恭喜猜對了！」，並結束遊戲。

### 功能實作要求

*   必須使用 `import random` 模組中的 `random.randint(1, 100)` 來產生隨機數字。
*   需要使用 `while` 迴圈來讓遊戲可以重複進行，直到玩家猜對為止。
*   需要使用 `input()` 函式來取得玩家輸入的猜測。
*   記得使用 `int()` 將玩家輸入的文字轉換成數字才能進行比較。
*   使用 `if/elif/else` 條件判斷來比對數字，並給予對應的提示。

### 執行範例

```
猜一個 1 到 100 之間的數字： 50
太小了！
猜一個 1 到 100 之間的數字： 75
太大了！
猜一個 1 到 100 之間的數字： 63
太小了！
猜一個 1 到 100 之間的數字： 68
恭喜猜對了！
```

### 如何執行

請在終端機中，確保路徑在專案資料夾底下，並執行以下指令：

```bash
python main.py
```
(假設您的主程式檔案名稱為 `main.py`)
